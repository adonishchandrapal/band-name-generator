# Band Name Generator 🎸

A simple Python project that generates a fun band name based on user input.

---

## 🚀 How It Works

1. The program asks for:
   - The name of the city you grew up in
   - Your pet's name
2. It combines both inputs
3. It generates a unique band name

---

## 🖥️ Example Run

```
What's the name of the city you grew up in?
Delhi
What's your pet's name?
Bruno

Your band name could be: Delhi Bruno
```
